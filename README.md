# SmartVideoInspector

AI-приложение для анализа видео производственных и ремонтных процессов.

## Запуск локально:

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Возможности:
- Анализ движения
- Графическая разметка зоны
- Поддержка пользовательских запросов
- Графики активности
- AI-рекомендации
